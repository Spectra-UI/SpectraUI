## Step 1: Backup Your WTF Folder

Before installing **SpectraUI**, it is highly recommended to start with a clean profile.

1. Go to your **World of Warcraft** installation folder.
2. Locate the **WTF** folder:
   - **Classic ERA / Anniversary:** ```\World of Warcraft\_Classic_Era_\WTF```
   - **MoP Classic:** ```\World of Warcraft\_classic_\WTF```
3. Copy and **store a backup** of this folder somewhere safe.

## Step 2: Enter the Game & Check AddOns

1. Launch **World of Warcraft**.
2. At the **Character Selection** screen, click **AddOns**.
3. Ensure all **required and recommended addons** are **enabled**.
4. If **SpectraUI** does not appear, enable **“Load out-of-date AddOns”**.

## Step 3: Run the SpectraUI Installer

Once inside the game:

1. The **SpectraUI installer** will automatically appear.
2. **Follow the on-screen instructions** carefully.
3. Adjust UI settings as desired.

> **If the installer does not appear, you can start it manually by typing:**  
> `/sui install` **in chat**

---

## Final Steps & Troubleshooting

If everything looks good — you’re all set. Enjoy your new **SpectraUI**!

### If you run into issues:
- Join the [**Discord**](https://discord.gg/UxHZRpSwQz) and open a **support ticket** in the appropriate channel for your game version.
- Run **`/reloadui`** after installation or any configuration changes.
- Double-check that all **required addons** are installed and loaded correctly.
